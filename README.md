## Setup
```
npm install
```

### Install SASS-loader
```
npm install --save-dev node-sass sass-loader
```

### Install vue-select
```
npm install vue-select
```

### Install axios
```
npm install axios
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
